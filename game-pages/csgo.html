Counter Strike Global Offensive

Counter-Strike: Global Offensive იგივე CS:GO — მრავალმოთამაშიანი პირველი პირის მსროლელი ვიდეო თამაში, შექმნილი Hidden Path Entertainment-ის და Valve Corporation-ის მიერ. მისი კომპოზიტორია მაიკ მორასკი. თამაში არის რიგით მეოთხე Counter-Strike-ის სერიაში. იგი გამოვიდა Microsoft Windows-ის, OS X-ის, Xbox 360-ის და PlayStation 3-ის პლატფორმებისთვის, 2012 წლის აგვისტოში, ხოლო Linux-ის ვერსია ოფიციალურად 2014 წლის სექტემბერში გამოვიდა. თამაშში ორი გუნდი ერთმანეთის წინააღმდეგ თამაშობს - ტერორისტიები და პოლიციელები. ტერორისტები ცდილობენ პოლიციელები დაბომბვას და მძევლების აყვანას, ხოლო პოლიციელები – მათ განეიტრალებას.
